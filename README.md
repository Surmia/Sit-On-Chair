# Sit-On-Chair

Furniture shop's website.

Demo version can be seen here: https://surmia.github.io/Sit-On-Chair/

## Functionalities:

* Dropdown menu
* Slider
* Highlighted subscription plan on 'mouseover' event
* Cost calculator

## Technologies used:

* HTML5
* CSS3
* JavaScript
* Sass
* Gulp
